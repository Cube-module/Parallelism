## Описание

Генерируется массив из 10⁷ элементов со значениями синуса от 0 до 2π. 
Вычисляется сумма всех элементов и выводится в терминал.

## Точность Float

-0.0277862

## Точность Double

6.27585e-10

## Сборка и запуск

### Требования
- CMake 3.14+
- Компилятор C++17 (MinGW или MSVC)

### Сборка

```bash
# Клонирование репозитория
git clone <url-репозитория>
cd sinus_hw_1

# Конфигурация с double (по умолчанию)
cmake -B build

# Или конфигурация с float
cmake -B build -DUSE_FLOAT=ON

# Сборка
cmake --build build
